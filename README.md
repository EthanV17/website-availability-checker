# Python Website Availability Checker

A simple Python script that checks whether a website is reachable.

## Features
- Sends an HTTP request to a website
- Detects whether the website is reachable
- Handles invalid URLs and connection errors
- Uses Python requests and exception handling

## Usage

python websiteuptime.py

Enter a website URL when prompted.

## Skills Demonstrated
- Python scripting
- HTTP requests
- Exception handling
- Basic network troubleshooting
